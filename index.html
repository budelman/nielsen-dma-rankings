Nielsen-dma-rankings

Playing around with d3.js and the nielsen dma rankings

View on GitHub Download .zip Download .tar.gz

This is the map of Nielsen Designated Marketing Areas using D3.js. Hover over an area for the corresponding data related to each marketing area.

View live map

Credits

  • Original Javascript without rankings courtesy of Simon Zou
  • Map adapted from Mike Bostock's map example
  • Topojson made by converting shapefile from geocommons to topojson via Mike Bostock's instructions
  • Nielsen rank and TV data via tvb

Notes

  • Some areas are shaped really strangely (such as a slice of Nevada being part of the Denver media market) but that is really how they are divided
  • There seems to be some mishandling of the data where Rochester, MI and Rochester, NY seem to have had their location data swapped.
